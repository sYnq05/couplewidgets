App-Icon für den Release

Für den App-Store-Release musst du hier mindestens ein Icon hinzufügen:

• Icon-iOS-Default-1024x1024@1x.png  (1024 × 1024 Pixel, iOS)

Optional (laut Contents.json):
• Icon-iOS-TintedDark-1024x1024@1x.png  (Dark Mode)
• Icon-iOS-TintedLight-1024x1024@1x.png (Tinted Light)

Ohne mindestens das 1024×1024-Bild baut das Archiv/Upload fehl oder wird abgelehnt.
